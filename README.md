# WanderlustMain
# 🌍 WanderLust – Airbnb Clone

A full-stack Airbnb-like web application built using the MERN stack with real-time listings, authentication, and booking features.

---

## 🚀 Live Demo

🔗 Frontend: https://wanderlust-main-alpha.vercel.app
🔗 Backend API: https://wanderlustmain.onrender.com

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Deployment:** Vercel (Frontend), Render (Backend)

---

## ✨ Features

* 🔐 User Authentication (Login/Signup)
* 🏡 Create & Manage Listings
* 📍 View Property Details
* 📅 Booking System
* 🧾 Host Dashboard
* ⚡ Real-time API Integration
* 🌐 Fully Deployed (Cloud)

## ⚙️ Installation (Local Setup)

```bash
# Clone repo
git clone https://github.com/your-username/WanderlustMain.git

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Run backend
cd ../backend
npm start

# Run frontend
cd ../frontend
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file in backend:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
PORT=5001
```
